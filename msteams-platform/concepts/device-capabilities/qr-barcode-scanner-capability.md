## <a name="code-snippet"></a><span data-ttu-id="4b371-203">Extrait de code</span><span class="sxs-lookup"><span data-stu-id="4b371-203">Code snippet</span></span>

<span data-ttu-id="4b371-204">**Appel `ScanBarCode()` API pour** analyser la QR ou le code-barres à l’aide de l’appareil photo :</span><span class="sxs-lookup"><span data-stu-id="4b371-204">**Calling `ScanBarCode()` API** for scanning QR or barcode using camera:</span></span>

```javascript
const config: microsoftTeams.media.BarCodeConfig = {
  timeOutIntervalInSec: 30};
microsoftTeams.media.scanBarCode((error: microsoftTeams.SdkError, decodedText: string) => {
  if (error) {
    if (error.message) {
      output(" ErrorCode: " + error.errorCode + error.message);
    } else {
      output(" ErrorCode: " + error.errorCode);
    }
  } else if (decodedText) {
    output(decodedText);
  }
}, config);
```
