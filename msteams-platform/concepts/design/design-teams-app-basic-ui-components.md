# <a name="designing-your-microsoft-teams-app-with-basic-fluent-ui-components"></a>Conception de votre application Microsoft teams avec des composants de base de l’interface utilisateur Fluent

Les applications teams sont généralement basées sur les composants de base de l’interface utilisateur Fluent suivants. Conçus de manière aussi simple que possible, ces composants peuvent fonctionner dans différents cas d’utilisation, thèmes et tailles d’écran.

Les illustrations de cette page indiquent l’apparence des composants dans les thèmes teams par défaut (Light) et Dark.

## <a name="microsoft-teams-ui-kit"></a>Kit d’interface utilisateur Microsoft teams

En fonction de l' <a href="https://fluentsite.z22.web.core.windows.net/" target="_blank">interface utilisateur Fluent</a>, le kit d’interface utilisateur Microsoft teams inclut des composants et des modèles conçus spécifiquement pour la création d’applications Teams. Dans le kit d’interface utilisateur, vous pouvez saisir et insérer les composants répertoriés ici directement dans votre conception et voir d’autres exemples d’utilisation de chaque composant.

> [!div class="nextstepaction"]
> [Obtenir le kit d’interface utilisateur Microsoft Teams (Figma)](https://www.figma.com/community/file/916836509871353159)
